# Killuma 🗡️

Luma who?

Killuma is the next-generation AI video forge designed to **kill** the competition. Built for creators who want:
- Cinematic, physically plausible videos
- Strong motion control & camera paths
- Image-to-video that actually respects your input
- Long, consistent clips (no random stuttering)
- 3D scene reconstruction from video
- Decentralized training so the model gets stronger the more people use it

### Core Features
- **Kill Mode** — One prompt → brutal cinematic output
- **Studio** — Timeline editor with motion brush, camera controls, extensions
- **Capture** — Upload phone videos → instant 3D Gaussian splat / NeRF-style scene
- **Swarm Training** — Run a node, contribute GPU, get rewarded in $KILL tokens
- Runs on top of open models (CogVideoX, Mochi, etc.) + custom "Kill LoRAs"

### Tech Stack
- Frontend: Next.js 15 + Tailwind + shadcn
- Backend: FastAPI + ComfyUI workflows
- AI: Diffusion + ControlNets + IP-Adapter + Temporal consistency hacks
- Decentralized: Bittensor/Petals style swarm + incentive layer

We don't ask for permission. We generate.

**Current status**: Early MVP — already generating better motion than base Luma in many cases.
